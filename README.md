# 🐍 Snake Game

A classic Snake game built using vanilla JavaScript and the HTML5 Canvas API.

👉 Play it here: [https://yourusername.github.io/repo-name/](https://rubenkc.github.io/SnakeGame/)

---

## Features

- Smooth snake movement with keyboard controls
- Collision detection (walls and self)
- Food system with dynamic snake growth
- Score tracking
- Pause and resume functionality
- Game reset on collision

---

## Controls

- Arrow Keys — Move snake

---

## Tech

- HTML5
- CSS3
- JavaScript (Canvas API)

---

## How it works

The game runs on a loop that continuously updates the snake’s position and re-renders the canvas.
State is used to manage movement, collisions, scoring, and game status (running, paused, game over).
